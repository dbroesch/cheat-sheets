pro tips
Filter self-osc: Resonance max on BPF/HPF → pure sine, blocks osc. Use as extra oscillator.
Audio-rate LFO: LFO at 100 Hz → Pitch via matrix = FM-style harmonics.
Complex envs: CycEnv (LOOP) → Env Attack/Decay = evolving ADSR. CycEnv → its own Amount = circular self-mod.
Freeze Spice/Dice: Set Arp + Spice/Dice, then Shift+A or B to freeze into sequence permanently.
Chord progressions: Chords osc + Key/Arp→Wave (Random mode) = auto-harmonization.
Paraphonic + Seq: Hold 2 keyboard notes → seq uses remaining 2 voices for countermelody.
DAW integration: Local OFF → keys/knobs → MIDI only → DAW sends back. No double-triggering.
Meta-modulation: Use CycEnv to modulate the amount of LFO→Pitch for vibrato depth control.
