quick tips

dim7 symmetry

dim7 chords repeat every 3 semitones. Only 3 unique dim7 shapes exist:
C°7 = E♭°7 = G♭°7 = A°7
C#°7 = E°7 = G°7 = B♭°7
D°7 = F°7 = A♭°7 = B°7

aug symmetry

aug triads repeat every 4 semitones. Only 4 unique shapes:
Caug = Eaug = G#aug
C#aug = Faug = Aaug
Daug = F#aug = A#aug
E♭aug = Gaug = Baug

dom7 → resolution

G7 resolves to C · D7 resolves to G · A7 resolves to D (dominant → tonic, a 5th down)
