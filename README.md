## Format

- **Layout:** US Letter, landscape, 2 pages
- **Font:** IBM Plex Mono
- **Files:** Each product has an `.html` source and a `.pdf` for printing
- **Style:** Inspired by the Teenage Engineering product guide aesthetic

## Printing

Download the PDF for any sheet and print at 100% scale on US Letter paper, landscape orientation. The layout is designed to fit on two sides of a single sheet.

## Official Documentation

Each sheet links to the official product guide. Full documentation is at [teenage.engineering/guides](https://teenage.engineering/guides).
